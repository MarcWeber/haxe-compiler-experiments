The haXe Programming Language

Using the compiler

The haXe compiler is a commandline compiler that can take the following parameters :

  • -cp (path) : adds a search path for files. Can be used several times for adding several different paths.
  • -v : verbose mode.
  • ...more later

The current version of the compiler does only typing, you can only play with it but it will not generate any binary right now

To compile the class my.Class implemented in the file c:\work\my\Class.hx then simply run : 

    haxe -cp "c:\work" my.Class

Please notice that the haXe compiler only takes class names as input and not file names.
