The haXe Programming Language

Download

The following downloads are available :

  • haXe compiler Alpha 1 for Windows

The alpha of the compiler does only typing, you can only play with it but it will not generate any binary right now

You can also access haXe sources CVS repository using the following : 

    cvs -d:pserver:anonymous@cvs.motion-twin.com:/cvsroot login
    (empty password)
    cvs -s:pserver:anonymous@cvs.motion-twin.com:/cvsroot co haxe

haXe sources are available under the GPL license.

If you want to compile from sources, please download install.ml, install OCaml, and simply run ocaml install.ml from the commandline line. After that you'll get a bin/haxe binary. You need also the directory haxe/std that contains all the standard haXe library.
